### Technologies

- Socket.io
- TypeScript/Node JS/Express
- MongoDB
- Agenda
- AWS S3 Bucket
- Nodemailer
- Swagger

## Pre-requisites

- You should have the following to run this project locally:
- - Node JS
- - TypeScript
- - MongoDB
- - S3 Bucket
- - Email Service Provider (Ex: gmail, in order to be able to send emails using nodemailer).
